HTML FORMATTING Elements

            Formatting elements were designed to display special types of text:

            < b> - Bold text---- The text will become Bold.
            < strong> - Important text---- The text will typically become Bold.
            < i> - Italic text---- The text will be Italic.
            < em> - Emphasized text---- The text will be typically Italic.
            < mark> - Marked text---- The text inside this tag will be highlighted.
            < small> - Smaller text---- This will make the text smaller than the other texts.
            < del> - Deleted text---- This will strike through the text.
            < ins> - Inserted text---- This will make underline on the text.
            < sub> - Subscript text---- This will make the text appear half a character below the normal line.
            < sup> - Superscript text---- This will make exponent to the text.

            EXAMPLE: 

            This text is bold
            This text is important!
            This text is italic
            This text is emphasized
            This is some smaller text.
            
Do not forget to buy milk today.
