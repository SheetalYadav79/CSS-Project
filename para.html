PARAGRAPH TAG 

            The HTML < p > element defines a paragraph.

            A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after a paragraph.
            
           EXAMPLE:

This is a paragraph.

This is a paragraph with extra spaces but the browser will only take one space after the words.Becoz the content is enclosed within the paragraph tag.



HTML Styles

            The HTML style attribute is used to add styles to an element, such as color, font, size, and more.

            The HTML style attribute has the following syntax:
            Syntax:
            < tagname style="property:value;">< /tagname >

BACKGROUND COLOR:

            This property is used to set the background color of the web page.

            Syntax:
            < body background-color="blue"> 
            This will set the background color of the web page to blue.

TEXT COLOR:

            The CSS font-family property defines the font to be used for an HTML element:

            EXAMPLE: 

            < h1 style="font-size:300px;">This is a heading
            This will set the font size of the heading to 300 pixels.

TEXT ALIGN:

            The CSS text-align property defines the horizontal text alignment for an HTML element.
            It can take values- "center","left","right" or "justify".

            EXAMPLE: 

            < h1 style="text-align:center;">Centered Heading
            This will align the text given in the heading tag to center.
